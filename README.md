# Low-Complexity-Enhancement-Video-Coding
MPEG Low Complexity Enhancement Video Coding (LCEVC) is a video compression enhancement standard designed to improve compression efficiency while keeping encoding and decoding complexity low. Officially standardized as:  MPEG-5 Part 2 LCEVC ISO/IEC 23094-2. The technology was primarily developed by V-Nova.
